# sendop.js

A TypeScript library for sending user operations for ERC-4337 contracts.

```
npm install ethers sendop
```

- only support ethers v6

### Dev Commands

```sh
bun install
docker compose up -d

bun run test
bun test
bun test -t 'should validate hex strings correctly'

bun run build
```
